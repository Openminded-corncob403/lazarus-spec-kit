# Produto — FreePascal/Lazarus Spec-Kit

## Propósito

Este spec-kit fornece as regras, convenções e padrões para desenvolvimento de aplicações **FreePascal/Lazarus (Object Pascal)** com auxílio de assistentes de IA (Copilot, Cursor, Gemini). Ele garante que todo o código gerado acompanhe:

- **Lazarus / FreePascal Pascal Guide** (Identificadores claros, nomenclaturas T, I, F)
- **Princípios SOLID**
- **Clean Code Clássico** (Blocos curtos, métodos descritivos com tipagem estrita)
- **Arquitetura em Camadas** (Domain, Application, Infrastructure, Presentation via LCL/Intraweb)

## Público-Alvo

- Desenvolvedores FreePascal/Lazarus que utilizam IA no Visual Studio Code, Cursor ou IDE associada.
- Profissionais que buscam refatorar código legado para Clean Code Object Pascal moderno em FreePascal.
- Equipes migrando de Delphi pra Lazarus querendo estabelecer trilhos fortes (skills padronizados).

## Referências

- `AGENTS.md` na raiz do repositório compila as macro regras arquiteturais.
- Arquivos contextuais `*.md` fornecem diretrizes isoladas por domínio via IA (Horse, SQLdb, IntraWeb).
